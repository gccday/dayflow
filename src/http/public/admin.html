常见问题

为什么普通用户访问管理员接口会 403？

管理员接口仅接受 admin token，普通用户 token 会被权限中间件拦截返回 403。

二维码登录超时怎么办？

可在“签到账号管理”点击“二维码登录”重新拉起会话，或在二维码面板点击二维码图片刷新。

创建签到账号后建议怎么操作？

建议先绑定登录用户与推送通道，再进行二维码登录，最后启用自动任务。
